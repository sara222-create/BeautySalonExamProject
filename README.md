Beauty Salon Web Application

This project is an interactive web application developed as part of the Advanced Web Programming exam.
The application represents a beauty salon website where clients can browse available services and
book appointments online through an interactive form.

The system is built using HTML, CSS, JavaScript, PHP, and MySQL.
It includes frontend design, backend processing, database integration, and a secure admin dashboard.
The admin interface is used to visualize booking statistics in a read-only mode.

The project source code is hosted on GitHub.
A functional demonstration video and a detailed PDF report are provided in accordance with the exam requirements.
